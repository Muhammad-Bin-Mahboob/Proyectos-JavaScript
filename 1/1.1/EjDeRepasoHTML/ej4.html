• URL (Uniform Resource Locator): Es una dirección única que se utiliza para localizar un recurso específico en la web, como una página web o un archivo. La URL generalmente incluye el protocolo (como http o https), el nombre de dominio (como www.ejemplo.com) y, a veces, una ruta específica y parámetros adicionales.
  • HTTP (Hypertext Transfer Protocol): Es un protocolo de comunicación que se utiliza para la transmisión de datos en la web. Permite la transferencia de información entre un servidor web y un navegador, facilitando la carga de páginas web y la interacción con recursos en línea. La versión segura de HTTP es HTTPS, que incluye cifrado para proteger la privacidad y la integridad de los datos.
  • TCP/IP (Transmission Control Protocol/Internet Protocol): Es un conjunto de protocolos de comunicación que forma la base de Internet. TCP se encarga de asegurar que los datos se transfieran de manera fiable y en el orden correcto, mientras que IP se ocupa de la dirección y el enrutamiento de los datos a través de redes. Juntos, permiten la comunicación entre dispositivos en una red.
  • Internet: Es una red global de redes interconectadas que permite la comunicación y el intercambio de datos entre dispositivos en todo el mundo. Utiliza una infraestructura de servidores, cables, routers y otros equipos para transmitir información a través de diversos protocolos de comunicación.
  • W3C (World Wide Web Consortium): Es una organización internacional que desarrolla y mantiene estándares para la web. Fundada por Tim Berners-Lee, el W3C se encarga de promover la interoperabilidad y la accesibilidad en la web, estableciendo directrices y recomendaciones técnicas para mejorar la web para todos los usuarios.
  • HTML (Hypertext Markup Language): Es un lenguaje de marcado utilizado para crear y estructurar contenido en la web. HTML define la estructura de una página web mediante el uso de etiquetas y elementos, como encabezados, párrafos, enlaces y listas. Es fundamental para la construcción de páginas web y se utiliza en combinación con otros lenguajes como CSS y JavaScript para diseñar y mejorar la funcionalidad de los sitios web.
